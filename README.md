Quick Data Prototype
================


Getting Started
---------------

You'll need to create a secrets.yml with the following:

development:
  domain_name: example.com
  secret_key_base: CHANGEME
  mapbox: CHANGEME

Ruby on Rails
-------------

This application requires:

- Ruby 2.2.1
- Rails 4.2.3
